مدد: بینرز

تفصیل

بینرز وہ اشتہار ہیں جو صفحہ کے اوپری حصے میں دکھائے جاتے ہیں۔ وہ تصاویر یا فلیش فلمیں ہو سکتی ہیں، جیسے XOOPS بینر جو پہلے سے طے شدہ تنصیب کا حصہ ہے۔ XOOPS میں بینرز کا انتظام کرنے اور اشتہاری کلائنٹس کے لیے اکاؤنٹس بنانے کے لیے ایک بلٹ ان سسٹم ہے، جو کہ سسٹم ماڈیول -> بینرز میں پایا جاتا ہے۔ آپ ایک XOOPS سائٹ پر متعدد بینرز رکھ سکتے ہیں - جب بھی نیا صفحہ لوڈ ہوتا ہے تو ڈسپلے کا علاقہ ہر ایک فعال بینرز کے ذریعے گھومتا ہے۔

ہر بار جب کسی وزیٹر کو بینر دکھایا جاتا ہے تو اس کا شمار ایک 'تاثر' کے طور پر ہوتا ہے، جو بینرز کی عام تجارتی کرنسی ہے۔ کلائنٹ عام طور پر نقوش کی ایک خاص تعداد کے لیے ادائیگی کرتے ہیں - یعنی ان کے اشتہار کو ایک خاص تعداد میں ظاہر کرنے کے لیے۔

براہ کرم نوٹ کریں: اگر آپ نقوش کی تعداد کی بنیاد پر بینرز فروخت کرنے کے کاروبار میں ہیں تو آپ کو ایک اہم کام کرنا چاہیے: سسٹم ماڈیول -> ترجیحات -> عمومی ترتیبات پر جائیں اور 'آپ کا آئی پی ایڈریس' فیلڈ پر کریں (اگر آپ کے پاس ہے)۔ یہ آپ کے دوروں کو روک دے گا۔ سائٹ کو تاثر کی تعداد میں شامل کرنے سے۔ ایسا کرنے میں ناکامی کو کسی حد تک بے ایمانی سمجھا جا سکتا ہے (تاہم، یہ نوٹ کرتے ہوئے کہ بہت سے منتظمین کے پاس مستقل IP نہیں ہوتا ہے - جیسے کہ ڈائل اپ موڈیم استعمال کرنے والے)۔
