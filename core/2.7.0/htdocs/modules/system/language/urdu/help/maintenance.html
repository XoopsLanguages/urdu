مدد: دیکھ بھال

تفصیل

مینٹیننس آپ کی XOOPS انسٹالیشن کو برقرار رکھنے اور اسے صحت مند رکھنے میں آپ کی مدد کرنے کے لیے کئی بنیادی افعال فراہم کرتا ہے:

- کیشے فولڈر کو صاف کریں۔
- سیشن ٹیبل کو خالی کریں۔
- ڈیٹا بیس ٹیبلز کی دیکھ بھال انجام دیں (تجزیہ کریں، چیک کریں، اصلاح کریں، اور مرمت کریں)

آپ بیک اپ کے مقاصد کے لیے پورے ڈیٹا بیس یا انفرادی ٹیبلز کا "ڈمپ" بھی کر سکتے ہیں۔
