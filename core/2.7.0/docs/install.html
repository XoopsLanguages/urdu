XOOPS انسٹال کرنا

پہلی بار انسٹالیشن

پیش لفظ:

XOOPS ایک اوپن سورس ہے۔ آبجیکٹ اورینٹڈ ویب پبلشنگ سسٹم پی ایچ پی میں لکھا گیا ہے۔ کے لیے ایک مثالی ٹول ہے۔ چھوٹی سے بڑی متحرک کمیونٹی ویب سائٹس، انٹرا کمپنی پورٹلز، کارپوریٹ پورٹلز، ویبلاگز اور بہت کچھ تیار کرنا۔

XOOPS کی شرائط کے تحت جاری کیا گیا ہے۔ GNU جنرل پبلک لائسنس (GPL) ورژن 2 یا اس سے زیادہ، اور استعمال اور ترمیم کرنے کے لیے آزاد ہے۔ جب تک آپ GPL کی تقسیم کی شرائط کی پابندی کرتے ہیں یہ دوبارہ تقسیم کرنا مفت ہے۔

ضروریات

  • WWW سرور (اپاچی۔, این جی آئی این ایکس، IIS، وغیرہ)
  • پی ایچ پی 8.2 یا اس سے زیادہ، 8.4+ تجویز کردہ
  • مائی ایس کیو ایل 5.7.8 یا اس سے زیادہ، MySQL 8.4.5+ تجویز کردہ

انسٹال کرنے سے پہلے

  1. WWW سرور، پی ایچ پی اور ڈیٹا بیس سرور کو درست طریقے سے ترتیب دیں۔
  2. اپنی XOOPS سائٹ کے لیے ڈیٹا بیس تیار کریں۔
  3. صارف اکاؤنٹ تیار کریں اور صارف کو ڈیٹا بیس تک رسائی فراہم کریں۔
  4. ڈائریکٹریز اور فائلوں کو قابل تحریر بنائیں: %s
  5. حفاظتی تحفظات کے پیش نظر، آپ کو سختی سے مشورہ دیا جاتا ہے کہ نیچے کی دو ڈائریکٹریوں کو باہر منتقل کر دیں۔ دستاویز کی جڑ اور فولڈر کے نام تبدیل کریں: %s
  6. بنائیں (اگر پہلے سے موجود نہیں ہے) اور ڈائریکٹریز کو قابل تحریر بنائیں: %s
  7. اپنے براؤزر کی کوکی اور جاوا اسکرپٹ کو آن کریں۔

مقامی طور پر انسٹال کرنا

اگر آپ ترقی یا جانچ کے لیے مقامی ماحول چلا رہے ہیں، تو یقینی بنائیں کہ آپ کے پاس پچھلا ہے۔ ضروریات کو پورا کیا. ایک بار جب یہ ہو جائے تو، کے مندرجات کو کاپی کریں۔ htdocs directory (from the XOOPS distribution file) to the root document path of your web environment. Once the files are copied there, you can start the install by pointing you browser to "http://yoursite.com"۔ یہ انسٹال کرنے کا عمل شروع کر دے گا۔

میزبان پلیٹ فارم پر انسٹال کرنا

اگر آپ میزبان ماحول میں چل رہے ہیں، تو XOOPS فائلوں کو مقامی طور پر یا سرور پر کھولیں اگر آپ کے پاس ہے شیل (یعنی SSH) تک رسائی۔ ایک بار مکمل ہو جانے کے بعد، تمام XOOPS فائلوں کو سے منتقل یا کاپی کرنا یقینی بنائیں htdocs آپ کی روٹ ویب ڈائرکٹری میں ڈائرکٹری (آپ کا فراہم کنندہ عام طور پر اس مقام کو ہدایات کے ساتھ فراہم کرتا ہے)۔ ایک بار فائلیں وہاں کاپی ہو جانے کے بعد، آپ اپنے براؤزر کی طرف اشارہ کر کے انسٹال شروع کر سکتے ہیں۔ "http://yoursite.com"۔ یہ انسٹال کرنے کا عمل شروع کر دے گا۔
